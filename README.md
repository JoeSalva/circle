# Social Media API

A scalable backend for a social media platform built with **Django** and **Django REST Framework**.  
This project implements core social networking features such as user authentication, post creation, likes, comments, and feed interactions through a structured RESTful API.

The goal of this project is to demonstrate production-style backend development, including clean architecture, relational data modeling, and secure API design.

## Features

- User registration and authentication
- Create, update, and delete posts
- Like and unlike posts
- Comment on posts
- User-based content ownership and permissions
- RESTful API endpoints built with Django REST Framework
- PostgreSQL-ready database structure

## Tech Stack

- **Python**
- **Django**
- **Django REST Framework**
- **PostgreSQL**
- **JWT / Token Authentication**

## Project Goal

This project was built as part of my journey to becoming a backend engineer, focusing on designing APIs that power modern social media platforms and scalable web applications.
